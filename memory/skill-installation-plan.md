# Skill Installation Plan
## AI项目经理制定 - 2026-01-31

### 侦察结果
- 发现672个可用skills
- 29个类别，覆盖我们所有需求
- 关键skills均存在（claude-team, conventional-commits等）

### 安装优先级

**Phase 1: 核心协作（立即执行）**
- `conventional-commits` - Dev必备，强制提交规范
- `github-pr` - Dev/QA必备，PR管理
- `claude-team` - PM必备（如果可用）

**Phase 2: 运维基础（今日完成）**
- `linux-service-triage` - Ops必备
- `deploy-agent` - Ops必备

**Phase 3: 增强能力（本周完成）**
- `agentlens` - HR/Analyst
- `frontend-design` - Editor
- 其他角色专用skills

### 执行策略
采用"快速试错"模式：
1. 并行尝试安装多个skills
2. 记录成功/失败
3. 对失败的稍后重试或标记为不可用

### 预期结果
- 核心skills安装率: 80%+
- 团队基础能力: 可用
- 阻塞问题: 识别并报告
