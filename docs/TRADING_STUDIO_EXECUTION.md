## 2. 快速验证机制

### 2.1 信号源分级验证

**阶段0：纸面验证（1-3天）**
```
新信号源发现
    ↓
Web Researcher收集历史信号
    ↓
Analyst回测验证（过去3个月数据）
    ↓
胜率>60%？→ 进入阶段1
胜率<60%？→ 淘汰
```

**阶段1：小资金试验（1-2周）**
```
试验层资金：单笔1-2万
    ↓
实盘跟踪记录
    ↓
连续10笔交易验证
    ↓
胜率>65% + 盈亏比>1.5？→ 进入阶段2
否则→ 淘汰或优化
```

**阶段2：中等仓位（2-4周）**
```
增长层资金：单笔3-5万
    ↓
连续20笔交易验证
    ↓
回撤<10%？→ 进入阶段3
否则→ 退回阶段1或淘汰
```

**阶段3：主力仓位（持续）**
```
核心层资金：单笔5-10万
    ↓
标准化执行
    ↓
每日监控，每月评估
    ↓
连续2个月亏损？→ 降级或淘汰
```

### 2.2 并行验证流水线

```
周1: 发现3个新信号源
    ↓
周2-3: 同时纸面验证3个
    ↓
周4: 1个通过→阶段1，2个淘汰
    ↓
周5+: 新信号源持续发现

目标：每月验证10个信号源，留下2-3个
```

## 5. 每日执行流程（老板12小时投入）

### 5.1 盘前（9:00-9:30）

```
09:00 木木推送《盘前情报简报》
        - 隔夜美股/加密动态
        - 当日重点监控信号
        - 持仓风险提示
        
09:15 老板审核
        - 确认当日交易计划
        - 调整资金配置
        
09:30 开盘执行
```

### 5.2 盘中（9:30-15:00）

```
实时监控（木木自动）
    ↓
信号触发 → 即时推送Telegram
    ↓
老板决策（2分钟内响应）
    ↓
执行/放弃
```

**紧急信号分级**:
- 🔴 一级（立即决策）：突破、暴跌、重大消息
- 🟡 二级（30分钟内）：一般信号、仓位调整
- 🟢 三级（收盘前）：复盘、明日计划

### 5.3 盘后（15:00-17:00）

```
15:05 自动生成《交易日报》
        - 当日盈亏
        - 信号命中率
        - 持仓风险分析
        
15:30 老板复盘
        - 决策质量回顾
        - 信号源评估
        
16:00 木木更新《信号源评分表》
        
17:00 Web Researcher开始收集晚间情报
```

### 5.4 晚间（20:00-24:00）

```
20:00 美股开盘监控
    ↓
22:00 《晚间情报汇总》
    ↓
23:00 老板制定明日策略
    ↓
24:00 系统夜盘监控（自动化）
```

## 7. 风险控制红线

### 7.1 绝对禁止

- ❌ 单票满仓（>10%仓位）
- ❌ 无止损持仓过夜
- ❌ 追涨杀跌（信号后5分钟内未决策放弃）
- ❌ 情绪交易（连续3笔亏损后强制休息1天）
- ❌ 杠杆融资（只用自有资金）

### 7.2 自动熔断

```
单日亏损达2% → 当日停止交易
单周亏损达5% → 降级到试验层
单月亏损达10% → 全仓止损，重新评估
```

- 木木检测到异常波动
- 多信号源同时预警
- 外部重大事件（政策、黑天鹅）
