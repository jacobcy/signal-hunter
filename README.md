# Signal Hunter Workspace

Signal Hunter is an automated signal hunting and reporting system focused on multi-source sourcing, analysis, and alerting.

## Quick Pointers
- Project structure: `STRUCTURE.md`
- Long-term memory: `MEMORY.md`

- Roadmap & backlog: `docs/project/ROADMAP.md`, `docs/project/BACKLOG.md`
- Code: `src/`
- Scripts: `scripts/`
- Tests: `tests/` (run via `./run-tests.sh`)
